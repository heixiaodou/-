# 1.Python安装
-----
## Python官方网址：
http://www.python.org/

点击链接我们会自动跳到Python官网，它会自动识别我们电脑的操作系统，所以网页显示的就是Python最新版本，直接点击下载就可以了。下载到电脑之后，双击安装包就进入了安装过程。
